# Codeforces-Solution

本项目作为个人刷题的记录，以及题解和代码的存储空间。

每个比赛目录下的 README.md 有这场比赛中每道题需要的知识点，在每个题目的 README.md 里是每道题的题解和代码。

To whom it may concern: 在补题时，先尝试自己思考，再确认有没有遗漏的知识点，在补充完知识点之后再尝试自己思考，最后再看题解。能不看额外信息自己做就自己做。
