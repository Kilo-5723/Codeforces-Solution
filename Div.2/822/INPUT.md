# Problem A

Solved at 00:03 (+)

题意：

给出 $n$ 根棍子，可以花费 $1$ 的代价将棍子的长度增加或减少 $1$，求最少花费多少代价才能用三根棍子构造出一个等边三角形。

题解：

将棍子按长度排序，最终选择的三根棍子一定是相邻的三根。同时将长度为 $a<b<c$ 的三根棍子变成长度相同需要的代价是 $c-a$，所以答案是 $\min\{a_{i+2}-a_i\}$。
